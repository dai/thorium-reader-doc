Thorium Reader est le lecteur EPUB de référence pour Windows, MacOS et Linux.

Cette application EDRLab est en constante évolution et constitue la référence pour accéder aux publications numériques EPUB 3 adaptable ou à mise en page fixe, aux livres audio et récits visuels, aux documents PDF et aux ebooks DAISY ; protégés LCP ou non.

Il est localisé dans un grand nombre de langues, chaque version offrant de nouvelles localisations.

Des efforts considérables sont également faits pour que Thorium Reader soit hautement accessible aux personnes aveugles, malvoyantes ou dyslexiques.

Cette application est gratuite, sans publicité et ne collecte pas de données privées. C'est l'outil idéal pour les gros lecteurs, les usagers des bibliothèques et les étudiants.

La dernière version de Thorium Reader est la 2.2.0, publiée en décembre 2022. * Version Windows 11 sur le Windows Store. * Version Windows sur la plateforme Github. * Version Mac OS sur la plateforme Github. * Linux (.AppImage) sur la plateforme Github, compatible avec de nombreuses distributions dont Ubuntu 20.04 et plus récentes (merci de nous dire sur lesquelles il fonctionne parfaitement ou non). * Linux (.deb) version sur la plateforme Github, compatible avec les distributions de type Debian, y compris Ubuntu 20.04 et plus récentes (nécessite GLIBC 2.28+).

* Affichage des métadonnées d'accessibilité. * Historique de navigation avec les boutons Précédent/Avant. * Raccourcis clavier pour le zoom sur EPUB Fixed-Layout. * Pages centrées sur EPUB Fixed-Layout. * Prise en charge de l'accessibilité MathML via MathJax. * Prise en charge du flux OPDS du serveur de contenu Calibre, avec authentification. * Possibilité d'éditer le titre et l'URL d'un flux OPDS. * Mise à jour des traductions : Anglais, chinois simplifié, chinois traditionnel, italien, japonais, lituanien, espagnol, suédois.

Si vous ne parvenez toujours pas à résoudre le problème, veuillez ouvrir une **Issue** (nécessite un compte GitHub) afin que nous puissions l'aborder et suivre la solution. Si vous ne pouvez pas ouvrir une **Issue** Github pour une raison quelconque, vous pouvez également envoyer un email à thorium-support at edrlab.org (@ remplacé pour éviter les spams), nous ferons de notre mieux pour vous répondre.
