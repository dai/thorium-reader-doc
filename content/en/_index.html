Thorium Reader is the EPUB reader of choice for Windows 10 and 11, MacOS and Linux.

This EDRLab application is in constant development and aims at becoming a reference for accessing EPUB 3 publications in reflow or fixed layout format, audiobooks and visual narratives, PDF documents and DAISY 2.02 and 3 accessible ebooks; LCP protected or not.

It is localized in a large set of languages, each version offering new locales.

Huge efforts are also made to get Thorium Reader highly accessible for visually impaired and dyslexic people.

This application is free, with no ads and no leaks of private data.

This is the perfect tools for heavy readers, library patrons and students.

The latest version of Thorium Reader is 2.2.0, released in December 2022.

  • Windows 11 version on the Windows Store.
  • Windows version on the Github platform.
  • MacOS version on the Github platform.
  • Linux (.AppImage) on the Github platform, compatible with multiple distributions including Ubuntu 20.04 and newer (please tell us on which it works perfectly or not).
  • Linux (.deb) version on the Github platform, compatible with Debian type distributions, including Ubuntu 20.04 and newer (requires GLIBC 2.28+).
{{% /blocks/feature %}} {{% blocks/feature icon="fa-solid fa-clipboard" title="2.2 new features" url="https://github.com/edrlab/thorium-reader/releases/tag/v2.2.0" %}}
  • Display of accessibility metadata.
  • Navigation history with back / forward buttons.
  • EPUB Fixed-Layout zoom keyboard shorcuts.
  • Single centered pages on EPUB Fixed-Layout.
  • MathML accessibility support via MathJax.
  • Support of the Calibre content server OPDS feed, with authentication.
  • Possibility to edit title and URL of an OPDS feed.
  • Updated translations: English, Simplified Chinese, Traditional Chinese, Italian, Japanese, Lithuanian, Spanish, Swedish.
